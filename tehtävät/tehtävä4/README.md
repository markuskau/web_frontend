# Tehtävä 4

### Tein testauksen, jossa robot framework suorittaa uuden entryn lisäämisen eli uuden päiväkirjamerkinnän lisäämisen.

### Tehtävässä tarkoitus on selvittää seuraavien kohtien avulla toimiiko entryn lisääminen oikein:
- Dialogi avautuu Add a new diary entry -napista
- Onnistunut merkinnän lisääminen kaikilla kentillä
- Dialogi suljetaan Close-napilla ilman tallennusta
- Kaikki lomakekentät löytyvät dialogista
- Kenttiin voi kirjoittaa tietoja
