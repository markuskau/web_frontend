# Tehtävä 1

## Asensin ohjeiden mukaisesti virtuaaliympäristön ja seuraavat paketit

-  ### Robo Framework asennus:
```bash
pip install robotframework
```
- ### Browser library asennus:
```bash
pip install robotframework-browser
```

- ### Alustin kirjaston toiminnan:
```bash
rfbrowser init
```

- ### Requests libraryn asennus:
```bash
pip install robotframework-requests
```

- ### Cryptolibraryn asennus:
```bash
pip install --upgrade robotframework-crypto
```

- ### Robotidyn asennus:
```bash
pip install robotframework-tidy
```

- ### Asennus listan tarkistus:
```bash
pip freeze
```

- ### Loin requirements.txt-asennuslistan:
```bash
pip install -r requirements.txt
```

- ### Lopuksi suoritin asennustesti.py tiedoston ja tulos oli tämä:
```bash
Robot Framework: 7.4.2
Browser: 19.9.0
requests: 2.32.5
CryptoLibrary: 0.4.2
```
